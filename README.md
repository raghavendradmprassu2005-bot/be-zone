# Be-Zone

Be-Zone is a modern beauty salon and spa booking platform built with React, TypeScript, Vite, Tailwind CSS, and Supabase.

## Features

- User Authentication
- Online Appointment Booking
- Service Catalog
- Responsive Design
- Admin Dashboard
- Secure Database Integration

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase
